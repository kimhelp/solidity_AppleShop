환불

mapping

1. buy
2. sell
3. getter

tx{
from : account
to: CA ETH
data: 0x020202
value: 1ETH
}

value를 넣으려면 payable 속성이 필요!

사과 사는 사람 = CA => 판매자

address에 payable 속성이 붙으면 객체처럼 여러 기능을 쓸 수 있음.
